Backgound

The Covid-19 pandemic marked a significant event in modern history, reshaping the world in unprecedented ways. First documented in Wuhan, China in late 2019, it rapidly spread across the globe, causing widespread mortality and economic disruption. Malaysia recorded over 5 million cases and significant loss of life by June 2024.

Impact to the Society

The economic impact was profound, leading to job losses and financial strain for many. In response, the Malaysian government permitted withdrawals from the Employees Provident Fund (EPF), to alleviate financial hardships. This resulted in a substantial RM145 billion being withdrawn by 8.1 million members, reflecting the scale of the crisis and the desperate measures people took to cope with the economic fallout.

The Problem We Face Today

As a result, 6.62 million (52% of the total) EPF members aged under 55 had savings of less than RM10,000. Despite numerous warning by economic experts on the downside of EPF withdrawals, we are seeing increased voices from the public calling for more withdrawals to be allowed citing inability to recover from the economic crisis.

The Madani Government, in bid to strike balance has announced recent restructuring into three accounts, including the Akaun Fleksibel or EPF Account 3, allows for more flexible access to funds.

To Educate

The issue in hand bears much important to the Author. Having stood at the same crossroad just as many others, the Author had decide between a stable post-retirement life, or to withdraw from EPF to fund wedding expenses. Despite the pressure, Author made a firm decision not to withdraw

The Author aims to educate the public on the importance of post-retirement savings, securing financial stability in one's golden years. Withdrawing from EPF Account 3 may provide short-term relief but can lead to a lower dividend rate and reduced savings, potentially compromising one's financial security during retirement.

How?

Through a simple calculator, we will calculate the savings or investments might grow over time using the power of compound interest.

We will also to calculate the potential lost one might incur if they withdraw amount before the age of 55

JavaCode

Here we will take a closer look on the functional Java code of the compound interest calculator.

  import java.util.Scanner;

  class Compound_Interest_Calculator

  {

   public static void main(String[] args)

    {

      Scanner input = new Scanner(System.in);

      System.out.print("Enter current Age: ");

      int CurrentAge = input.nextInt();

      int CompoundYears = 0;

      int WithdrawnAge = 0;

      int Duration = 0;

      double InterestLost = 0;

      double WithdrawnAmount = 0;


      if (CurrentAge >=55)

        {

        System.out.print("You have reached Withdrawal Age");

        }

      else

        {

        System.out.print("Please enter principal amount: ");

        double Principal = input.nextDouble();

        CompoundYears = (55- CurrentAge + 1);

        double Rate = (5.5/100);


        System.out.print("Do You wish to withdraw amount before age 55?");

        char Withdraw = input.next().charAt(0);


      if (Withdraw == 'Y'|Withdraw == 'y')

        {

            System.out.print("At what age would you like to withdraw?:");

            WithdrawnAge = input.nextInt();

            System.out.print("Enter the amount to withdraw: ");

            WithdrawnAmount = input.nextDouble();

            Duration = (55- WithdrawnAge + 1);

            InterestLost = LostCalculation(Duration, WithdrawnAmount);

        }


      double InterestEarned = Principal;

      for(int year = 1; year <= CompoundYears; year++)

        {

            InterestEarned *= (1 + Rate);

        }


      System.out.println("Interest Rate (%): " + Rate*100);

      System.out.println("Compound Interest Earned by Age of 55: RM" + Math.round((InterestEarned-Principal)*100.0)/100.0);

      System.out.println("Final Savings Amount by Age of 55: RM" + Math.round((InterestEarned-InterestLost)*100.0)/100.0);

      System.out.println("Withdrawn Amount: RM"+ + Math.round(WithdrawnAmount*100.0)/100.0);

      System.out.println("Compound Interest Lost by Age of 55: RM" + Math.round((InterestLost-WithdrawnAmount)*100.0)/100.0);    

        }

      }

   public static double LostCalculation(int Duration, double WithdrawnAmount)

   {

        double result = WithdrawnAmount;

        double Rate = (5.5/100);

        for (int year = 1; year <= Duration; year++)

        {

        result *= (1 + Rate);

   }

   return result;

    }

  }
